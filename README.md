# Parking System

This project contains the source code for the Spread Fullstack Developer bootcamp challenge.

### Technologies

- Vite
- Scss
- EditorConfig
- Husky
- Typescript

### How to install?

**Clone the project and access the folder**

```
$ git clone https://github.com/dcaiovinicius/instagram-login-clone
```

```bash

$ yarn

$ yarn dev

```
